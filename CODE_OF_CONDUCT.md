# Code of Conduct

TerritoryKit uses the Contributor Covenant Code of Conduct, version 2.1, as the baseline for
community participation.

Be respectful, constructive, and specific. Harassment, abuse, and bad-faith behavior are not
accepted in issues, pull requests, discussions, or project spaces.

Project maintainers may remove comments, close issues, or block participants when needed to
keep the project usable and safe.
