# Administrative Semantics

`datasets/registry/admin-semantics.json` is generated by `pnpm registry:generate` and uses schema
`territorykit-admin-semantics@2`.

The registry contains `ADM0` through `ADM5` for every ISO country or area. Reviewed mappings are
country-specific. Unknown mappings remain `semanticType: "unknown"` with
`mapping-review-required`/`review-required` status until reviewed.

Turkey currently has reviewed semantic mappings for:

| Level | Semantic Type | Local Type |
| ----- | ------------- | ---------- |
| ADM0  | country       | Ülke       |
| ADM1  | province      | İl         |
| ADM2  | district      | İlçe       |
| ADM3  | neighbourhood | Mahalle    |

Turkey has a partial official Gaziantep ADM3 artifact, but this is not a claim of complete official
Turkey ADM3 geometry coverage.
